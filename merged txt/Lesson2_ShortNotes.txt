
How to setup Bootstrap?

To setup Bootstrap, follow any of the below given method:

1. Download Bootstrap: Download from getbootstrap.com   
2. CDN (Content Delivery Network): jsDelivr is the official CDN of Bootstrap


!! Method 1: Download Bootstrap !!

Download Bootstrap from getbootstrap.com to host Bootstrap yourself.


!! Method2: CDN (Content Delivery Network) !!

CSS
<!-- Compiled and minified CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet">

JavaScript
<!-- Compiled JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/js/bootstrap.bundle.min.js"></script>
